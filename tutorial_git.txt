3.- Inicializar el repositorio

4.- Agregar archivos al repositorio
        @git add .
5.- Validar el estatus del repositorio (opcional)
        @git status
6.- Autentificar el usuario
        git config --global user.email "you@example.com"
        git config --global user.name "Your Name"
6.- Creacion de un commit
        @git commit -m'texto del commit'
